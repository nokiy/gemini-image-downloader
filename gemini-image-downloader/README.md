# Gemini Image Downloader

Gemini Image Downloader 是一款 Chrome 浏览器扩展，专为 Google Gemini 网页版设计，支持一键批量检测并下载对话中的 AI 生成图片。

## ✨ 主要功能

- **🚀 自动检测**：智能识别当前对话页面中加载的 AI 图片。
- **📸 悬浮图标**：在页面右下角显示检测到的图片数量，点击即可预览。
- **📦 批量下载**：支持全选或手动选择图片，一键打包下载为 ZIP 文件。
- **🖼️ 高清原图**：自动获取图片最高清晰度版本。
- **📂 智能命名**：自动处理文件名冲突，ZIP 包按 `Gemini_Images_{数量}.zip` 命名。

## 📥 安装说明

1. 下载源代码或 Release 包。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启右上角的 **"开发者模式"**。
4. 点击 **"加载已解压的扩展程序"**，选择 `gemini-image-downloader` 文件夹。

## 🎮 使用指南

1. 打开 [gemini.google.com](https://gemini.google.com/) 并开始对话。
2. 当 Gemini 生成图片后，页面右下角会自动出现蓝色悬浮图标。
3. 点击图标打开图片列表抽屉。
4. 勾选需要下载的图片（支持"全选"）。
5. 点击 **"批量下载"** 按钮，等待打包完成后即可保存。
6. 点击每张图片右下角的下载图标，可单独下载该图片。

## 🛠️ 技术特性

- **In-Page UI**：完全嵌入页面的原生风格 UI，操作更流畅。
- **Service Worker**：后台处理下载任务，支持大文件打包。
- **任务队列**：支持多任务排队，避免并发下载导致的卡顿。
- **隐私安全**：所有处理均在本地浏览器完成，不上传任何数据。

## 📋 常见问题

**Q: 为什么有时候图标不显示？**
A: 请确保页面已完全加载，且对话中确实包含 AI 生成的图片。如果图片正在生成中，请稍候片刻。

**Q: 下载失败怎么办？**
A: 请检查网络连接。如果是批量下载失败，尝试减少一次下载的数量。
